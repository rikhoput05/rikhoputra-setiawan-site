# Quick Web (Gratis)

Website cepat siap upload.

## Cara paling cepat online (gratis)

### Opsi A — Netlify Drop (1 menit)
1. Buka: https://app.netlify.com/drop
2. Drag & drop file `index.html` ke halaman tersebut.
3. Langsung dapat link web gratis.

### Opsi B — GitHub Pages
1. Buat repo baru di GitHub (mis. `web-ayah`).
2. Upload file `index.html`.
3. Di Settings → Pages, pilih branch `main` folder `/root`.
4. Tunggu 1-2 menit, web aktif di URL GitHub Pages.

## Edit cepat
- Ubah judul: tag `<title>`
- Ubah isi: teks di `<h1>` dan `<p>`
- Ubah warna: variabel CSS di `:root`
